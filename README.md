# Purpose of eureka-client
It is based on the [examples](https://github.com/Netflix/eureka/tree/master/eureka-examples) of [eureka-client](https://github.com/netflix/eureka),  to simplify the eureka client api 
The location of properties file is set by system parameter, to refer to the test case for detail.    
The [eureka server](https://github.com/Netflix/eureka/tree/master/eureka-server) is used in the test.
